# GTARenderHook
GTA SA rendering hook
Implements DirectX11(also there is some Vulkan API test files) rendering API to GTA San Andreas.
# Requirements to
1) Windows 7+
2) Latest DirectX SDK
3) Latest Vulkan SDK

# How to setup
1) Compile project in release mode.
2) Put output files and shaders folder to main directory of GTA SA
3) Run!
